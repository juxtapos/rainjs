Class PatternLayout¶

A layout that allows the administrator to configure the pattern for the logged messages:

"logger": {
    "level": "info",
    "appenders": [{
        "type": "console",
        "layout": {
            "type": "pattern",
            "params": {
                "pattern": "[%level] %date: %message"
            }
        }
    }
}

PatternLayout#format(event)¶
Arguments:
  • event (Object) – the log event
  • event.level (String) – the logging level
  • event.date (String) – the message date
  • event.name (String) – the logger name
  • event.error (RainError) – the error

:returns :
