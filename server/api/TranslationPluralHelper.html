Class TranslationPluralHelper¶

This Handlebars helper is used to translate plural text.

Syntax:

{{nt msgId msgIdPlural count [[arg1] [arg2] [argN]]}}

Examples¶

<span>
     {{nt "%1$s %2$s, you won one car." "%1$s %2$s, you won %3$d cars." 4 "Mr." "Wayne" 4}}
 </span>

Constructor¶

class TranslationPluralHelper()¶

Methods¶

helper¶

TranslationPluralHelper#helper(msgId, msgIdPlural, count)¶
Arguments:
  • msgId (String) – the message id for the translation
  • msgIdPlural (String) – the message id for the plural translation
  • count (String) – the count for the plural decision
  • arg1..N (String|Number) – the arguments for the translation text
Returns String:

translated text

Translates a message to the language specified in the platform configuration. First it checks if the message exists for the platform language. If this is not found it tries the default language. The last fallback is to return the message passed as parameter. It also decides if the singular or plural form should be used based on the count parameter.
