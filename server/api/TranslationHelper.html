Class TranslationHelper¶

This Handlebars helper is used to translate singular text.

Syntax:

{{t msgId [[arg1] [arg2] [argN]]}}

Examples¶

<span>
     {{t "How are you?"}}
     {{t "How are you %s?" "Batman"}}
 </span>

Constructor¶

class TranslationHelper()¶

Methods¶

helper¶

TranslationHelper#helper(msgId)¶
Arguments:
  • msgId (String) – the message id for the translation
  • arg1..N (String|Number) – the arguments for the translation text
Returns String:

the translated text

Translates a message to the language specified in the platform configuration. First it checks if the message exists for the platform language. If this is not found it tries the default language. The last fallback is to return the message passed as parameter.
