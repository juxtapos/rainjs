Class Event¶

Represents a logger event.

Constructor¶

class Event(level, message, error, logger, source)¶
Arguments:
  • level (Number) – The level of the event.
  • message (String) – The log message.
  • error (RainError) – The error associated with the log message.
  • logger (String) – the logger name
  • source (String) – the log statement location: ‘server’ or ‘client’
