Class User¶

Provides a basic implementation of the user class. It can serve as a base class for a custom user implementation if additional fields or methods are needed.

Constructor¶

class User(obj)¶
Arguments:
  • obj (Object|User) – contains the data used to initialize an user.
  • obj.id (String) – the user id
  • obj.username (String) – the username
  • obj.permissions (String[]) – the permissions associated with this user
