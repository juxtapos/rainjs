Class ProxyStore¶

Proxy store implementation for connect. Uses dependency injection to set the store implementation. By default, a MemoryStore() is used. It uses lazy instantiation.

Constructor¶

class ProxyStore([Store], cookie)¶
Arguments:
  • Store (connect.session.Store) – store implementation (Default: MemoryStore)
  • cookie (Object) – the cookie configuration for the session store
  • cookie.path (String) – the path to be set to the cookies
  • cookie.httpOnly (Boolean) – wether the cookie is http only or not
