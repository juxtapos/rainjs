Class ProxyStore
================================================================================

..
   class-title


Proxy store implementation for connect. Uses dependency injection to set the store implementation.
By default, a :js:class:`MemoryStore` is used. It uses lazy instantiation.








    


Constructor
-----------

.. js:class:: ProxyStore([Store], cookie)



    
    :param connect.session.Store Store: 
        store implementation (*Default*: MemoryStore)
    
    :param Object cookie: 
        the cookie configuration for the session store 
    
    :param String cookie.path: 
        the path to be set to the cookies 
    
    :param Boolean cookie.httpOnly: 
        wether the cookie is http only or not
